# Neural Network From Scratch (C++)

A from-scratch neural network implementation in C++ - no TensorFlow, no PyTorch.

This project demonstrates core deep learning concepts: **matrix math**, **forward propagation**, **backpropagation**, and **stochastic gradient descent (SGD)**.

## Features

| Component | Implementation |
|-----------|----------------|
| Matrix ops | Addition, subtraction, multiplication, transpose, broadcasting, element-wise ops |
| Activations | Sigmoid, ReLU, Softmax (+ derivatives) |
| Loss | MSE (XOR), Cross-entropy (MNIST) |
| Optimizer | Mini-batch SGD |
| Datasets | XOR (built-in), MNIST subset (100 samples/class) |

## Architecture

**XOR:** `2 → 8 (sigmoid) → 1 (sigmoid)` with MSE loss

**MNIST:** `784 → 128 (ReLU) → 64 (ReLU) → 10 (softmax)` with cross-entropy loss

## Build

Requires **C++14** and a compiler (MSVC, GCC, or Clang). CMake 3.16+ is optional.

### Option A: CMake

```bash
cmake -S . -B build
cmake --build build
```

### Option B: Direct compile (Windows / MinGW)

```bat
build.bat
```

Or manually:

```bash
g++ -std=c++14 -Wall -Wextra -Iinclude src/*.cpp ... -o build/train_xor
```

## Run

### XOR (no data download needed)

```bash
./build/train_xor
```

### MNIST subset

Generate the dataset first (downloads MNIST once, ~11 MB):

```bash
python scripts/generate_mnist_subset.py
```

Then train:

```bash
./build/train_mnist
```

Optional custom CSV path:

```bash
./build/train_mnist path/to/custom.csv
```

## Math Overview

**Forward pass** (per layer):

\[
Z = W A + b,\quad A = \sigma(Z)
\]

**Backpropagation** (chain rule):

\[
\frac{\partial L}{\partial W} = \delta A_{prev}^T,\quad
\frac{\partial L}{\partial b} = \sum \delta,\quad
\delta_{prev} = W^T \delta \odot \sigma'(Z_{prev})
\]

**SGD update:**

\[
W \leftarrow W - \eta \frac{\partial L}{\partial W}
\]
